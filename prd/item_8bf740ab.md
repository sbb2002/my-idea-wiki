# 기타 연주 스킬 개발 시스템 PRD

## Overview

기타리스트의 메이저 스케일 숙달과 악보 연주 능력 향상을 위한 체계적 학습 플랫폼. 6번줄 메이저 스케일의 포지션별 음이름 학습부터 시작하여, Eb 메이저 스케일 3번품 포지션 집중 훈련, 프렛 간격 암기, 최종적으로 오선지 악보 독보 연주까지의 단계별 커리큘럼을 제공한다.

이 시스템은 "계산 없이 바로 연주"할 수 있는 근육 기억과 이론적 이해의 완벽한 결합을 목표로 하며, 기존의 일반적인 기타 학습 앱과는 달리 **특정 스케일과 포지션에 극도로 집중**하여 완벽한 숙달을 추구한다는 점에서 차별화된다.

## Core Value

**6번줄 메이저 스케일과 Eb 메이저 3번품 포지션의 완벽한 암기를 통해, 기타리스트가 프렛 간격을 머리로 계산하지 않고 손가락이 자동으로 찾아가는 수준의 근육 기억을 구축하는 것**

## MVP Scope

### 포함되는 기능

1. **6번줄 메이저 스케일 포지션 학습 모듈**
   - 1번품(솔라), 2번품(라시도), 3번품(시도레), 4번품(도레미), 5번품(미파솔) 시각화
   - 각 포지션별 음이름 표시 및 반복 학습

2. **Eb 메이저 스케일 3번품 포지션 집중 훈련**
   - 11프렛 6번줄(Eb)을 '도'로 고정한 연습 환경
   - 3번품 포지션(시도레) 패턴 시각화 및 반복 훈련

3. **프렛 간격 암기 시스템**
   - 도 기준 음정 간격 표시 (도~레: 2프렛, 도~미: 4프렛, 도~솔: 7프렛, 도~라: 9프렛, 도~도: 12프렛)
   - 간격별 반복 훈련 및 암기 확인 퀴즈

4. **진도 추적 및 숙달도 측정**
   - 각 포지션별 연습 시간 및 정확도 기록
   - 암기 완성도 평가 시스템

5. **기본 오선지 악보 연주 연습**
   - 학습한 포지션 내에서 연주 가능한 간단한 악보 제공
   - 실시간 연주 정확도 피드백

### 명시적 제외 사항

- 다른 현(1-5번줄)의 스케일 학습
- Eb 메이저 외의 다른 조 학습
- 3번품 외의 다른 포지션 상세 훈련
- 복잡한 악보나 곡 연주 기능
- 코드 진행이나 리듬 패턴 학습
- 소셜 기능이나 커뮤니티 요소
- 실제 기타 연결을 통한 자동 인식 기능

## Requirements

### 기능 요구사항

1. **시각적 기타 프렛보드 인터페이스**
   - 6번줄 중심의 프렛보드 시각화 (최소 12프렛까지)
   - 각 프렛별 음이름 표시/숨김 토글
   - 현재 학습 중인 포지션 하이라이트 표시

2. **포지션별 학습 모드**
   - 5개 포지션(1-5번품) 개별 학습 모드
   - 각 포지션에서 음이름과 프렛 위치 매핑 연습
   - 순서대로 또는 랜덤으로 음이름 제시 후 프렛 찾기

3. **Eb 메이저 집중 훈련 모드**
   - 11프렛을 '도'로 고정한 상태 표시
   - 3번품 포지션(시도레) 패턴 반복 연습
   - 시도레 패턴에서 각 음의 상대적 위치 암기 훈련

4. **프렛 간격 암기 시스템**
   - 도 위치에서 시작하여 목표 음까지의 프렛 간격 계산 연습
   - 5개 핵심 간격(2, 4, 7, 9, 12프렛) 집중 훈련
   - 타이머를 활용한 즉석 반응 테스트

5. **진도 관리 시스템**
   - 각 학습 영역별 연습 시간 기록
   - 정답률 및 반응 속도 추적
   - 숙달 기준 달성 여부 표시 (예: 95% 정확도, 평균 반응시간 2초 이하)

6. **기본 악보 연주 모드**
   - 학습한 포지션 범위 내의 간단한 악보 제시
   - 오선지에서 음표 제시 시 해당 프렛 위치 찾기
   - 연주 정확도 및 속도 평가

### 사용자 인터페이스 요구사항

1. **직관적인 프렛보드 시각화**
   - 실제 기타와 유사한 비율과 색상 사용
   - 프렛 번호 및 음이름 명확한 표시
   - 현재 학습 중인 영역 시각적 강조

2. **단계별 학습 네비게이션**
   - 현재 학습 단계 명확한 표시
   - 이전/다음 단계 이동 버튼
   - 전체 학습 과정에서의 현재 위치 표시

3. **실시간 피드백 시스템**
   - 정답/오답 즉시 표시
   - 올바른 답안 시각적 제시
   - 연습 통계 실시간 업데이트

### 성능 요구사항

1. **반응 속도**
   - 사용자 입력 후 0.1초 이내 피드백 제공
   - 화면 전환 시 0.3초 이내 로딩 완료

2. **정확성**
   - 음이름-프렛 매핑 100% 정확성 보장
   - 프렛 간격 계산 오류 0%

## Technical Considerations

### 아키텍처 가정사항
- 웹 기반 애플리케이션으로 구현 (크로스 플랫폼 호환성)
- 클라이언트 사이드 데이터 저장 (로컬 스토리지)
- 서버리스 아키텍처 또는 정적 호스팅 가능

### 데이터 구조

1. **스케일 데이터**
```
포지션: {
  번호: 1-5,
  음이름_배열: ["솔", "라"] | ["라", "시", "도"] | ["시", "도", "레"] | ["도", "레", "미"] | ["미", "파", "솔"],
  프렛_시작: 해당_포지션_시작_프렛
}
```

2. **진도 데이터**
```
학습자_진도: {
  포지션별_정확도: {},
  프렛간격_숙달도: {},
  전체_연습시간: number,
  최근_세션: timestamp
}
```

### 구현 우선순위
1. 기본 프렛보드 시각화 및 음이름 표시
2. 6번줄 포지션 학습 모드
3. Eb 메이저 3번품 집중 훈련
4. 프렛 간격 암기 시스템
5. 진도 추적 기능
6. 기본 악보 연주 모드

## Risks & Assumptions

### 핵심 가정사항 (취약점)

1. **학습자가 표준 튜닝(EADGBE)의 기타를 사용한다는 가정**
   - 위험도: 중간
   - 대안: 다른 튜닝 사용 시 시스템이 무용지물이 됨

2. **6번줄 중심 학습이 전체 기타 연주 능력 향상에 충분하다는 가정**
   - 위험도: 높음
   - 이 가정이 틀릴 경우 학습자가 제한된 연주 능력만 갖게 될 수 있음

3. **Eb 메이저 스케일 하나의 완벽한 숙달이 다른 조로의 확장에 도움이 된다는 가정**
   - 위험도: 중간
   - 만약 조별 특수성이 크다면 제한적 학습 효과

### 기술적 위험요소

1. **프렛보드 시각화의 정확성**
   - 실제 기타와의 비율 및 간격 불일치 가능성
   - 해상도별 시각적 왜곡 문제

2. **학습 진도 측정의 객관성**
   - 숙달도 평가 기준의 주관성
   - 실제 연주 능력과 시스템 평가 간 괴리 가능성

### 사용자 경험 위험요소

1. **반복 학습의 지루함**
   - 동일한 패턴 반복으로 인한 학습 동기 저하
   - 완벽한 암기까지의 긴 학습 기간

2. **실제 기타 연주와의 차이**
   - 화면상 학습과 실제 기타 연주 간의 감각적 차이
   - 손가락 근육 기억 형성의 한계
